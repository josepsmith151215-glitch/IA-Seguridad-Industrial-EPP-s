# IA-Seguridad-Industrial-EPP-s
Optimiza la seguridad industrial con VisionGuard-SST, una solución de vanguardia basada en Inteligencia Artificial. Este sistema emplea visión artificial (YOLO y OpenCV) para detectar en tiempo real el uso correcto de EPP (cascos, chalecos y más). Ideal para entornos de alto riesgo, garantiza entornos laborales seguros y tecnificados.
